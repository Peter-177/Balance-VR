# Agent Rules

Before starting any task:

1. Read BALANCE_VR_SPEC.md completely.
2. Review all available skills in the agent folder.
3. Select the most relevant skill.
4. Create an implementation plan.
5. Verify acceptance criteria.
6. Begin implementation.

Rules:

- BALANCE_VR_SPEC.md is the source of truth.
- Never redesign the UI.
- Never change dimensions.
- Never change colors.
- Never modernize the design.
- Use specialized skills whenever applicable.
- Verify work before completion.
